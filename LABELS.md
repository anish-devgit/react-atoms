# Repo labels (human-readable)

- bug (red) — Something is broken / reproducible
- enhancement (cyan) — New feature or improvement
- good first issue (green) — Easy tasks for new contributors
- help wanted (teal) — We need help
- docs (blue) — Documentation
- performance (orange) — Performance work
- a11y (purple) — Accessibility
- ux (yellow) — User experience
- design (violet) — Design assets
- component (blue) — Component code changes
- background (deep purple) — Background/canvas work
- preview (mint) — Live preview/demo issues
- blocked (grey) — Blocked by external dependency
- wontfix, duplicate — meta labels
